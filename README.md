\# 📊 Smart Sales Intelligence Platform



> \*\*An Executive Business Intelligence Dashboard built with Python, Streamlit, Pandas and Plotly.\*\*



A modern, interactive Business Intelligence (BI) application designed to help businesses analyze sales performance, monitor KPIs, visualize trends, and generate actionable insights from sales data.

\### 📈 Executive Dashboard



\* Executive KPI Dashboard

\* Business Health Score

\* Professional UI

\# 🚀 Installation



Clone the repository:



```bash

git clone https://github.com/tejasnew0099-dot/smart\_sales\_intelligence\_platform.git

```



Move into the project folder:



```bash

cd smart\_sales\_intelligence\_platform

```



Install dependencies:



```bash

pip install -r requirements.txt

```



Run the application:



```bash

streamlit run dashboard/app.py

```
